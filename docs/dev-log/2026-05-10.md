- FastAPI는 ASGI 받는 쪽(앱), uvicorn은 호출하는 쪽(서버). ASGI 표준이 둘을 잇는 계약
- TestClient는 소켓/HTTP 없이 메모리에서 ASGI 시그니처로 app 직접 호출 → 서버 안 띄워도 테스트 가능
- `app` 객체는 메모리상 callable일 뿐. 프로세스 시작 시 .py 1번만 import 후 메모리에서 동작
